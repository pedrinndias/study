O Projeto

De Estudante para Comunidade.

O Organize nasceu para transformar a solidão do código em potência colaborativa. Desenvolvido como projeto final da disciplina de Desenvolvimento de Interfaces Web, esta plataforma conecta desenvolvedores aspirantes a projetos reais.

Como Usar a Plataforma

Criando sua Conta e Perfil

Ao clicar em "Entrar" e selecionar a aba "Registar", você cria seu acesso. O sistema gera automaticamente um Perfil Público.

Encontrando Equipes

Utilize a barra de pesquisa na Home para filtrar equipes. Clique no ícone de Coração para salvar nos seus Favoritos.

Criando e Gerenciando Projetos

Clique em "+ Equipe" para criar um projeto. Você se tornará o Responsável e terá controle total no Dashboard.
